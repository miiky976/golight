# Golight
Just a much more simple copy of acpilight but made in Go.

There are some missing things.

And it doesnt work as acpilight.
I just followed a bit the way it works.

# Usage

## Get the actual level
```
light
```
By default its gonna return the level in every execution except when using flags like `--help` or `--list` or when there's an err in the usage.

## Set
```
light --set 0
light -set 99
```

## Increase and Decrese
```
light --id 6
light --id -6
```

## Increase and Decrese by Percentage
```
light --idp 10
light --idp -10
```
